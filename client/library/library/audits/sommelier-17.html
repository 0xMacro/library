The security audit was performed by the Macro security team on March 1st, 2024.
  • Discussions with the {{page.clientName}} team.
  • Available documentation in the repository.

Specifically, we reviewed the Uniswap V3 adaptors compatibility with Pancakeswap V3 contracts, ensuring they can be used to add and remove liquidity positions as well as swap tokens on Pancakeswap V3 pools. It was concluded that it is safe to interact with Pancakeswap V3 pools using either the UniswapV3Adaptor or SwapWithUniswapAdaptor, as was done with a Uniswap V3 pool.
